## 这个平台到底是干什么的

99designs成立于2008年，总部在澳大利亚墨尔本，后来把美国办公室开到了旧金山湾区。截至目前，平台上有超过146万注册设计师，来自全球192个国家。这些设计师已经为超过44万客户完成了33万多个项目。

简单来说，99designs就是一个连接设计需求方和设计师的在线平台。不管你是需要Logo、网页设计、名片、海报、产品包装，还是T恤图案、书籍封面，这里基本都能搞定。

平台提供三种服务模式：

**设计竞赛模式**：这是99designs最有特色的服务。你发布一个设计需求，然后会有几十个设计师给你提交作品，你从中选择最满意的，只需要支付给获胜设计师。这种模式最大的好处是你能看到各种风格的设计方案，选择空间特别大。

**一对一项目**：如果你已经找到了喜欢的设计师，或者项目比较复杂需要深度沟通，可以选择这种模式。你可以浏览设计师的作品集，直接雇用他们。平台会收取5%的服务费和2.5%的支付处理费。

**99designs Pro**：这是针对设计公司和大企业的专业服务，年费500美元。会有专属客户经理帮你对接设计师，还能享受私密竞赛、加急服务等特权。

## 它跟国内的猪八戒有什么区别

可能有人会问，这不就是国外版的猪八戒吗？

其实区别还挺大的。猪八戒主要是自由报价模式，设计师看到项目后自己报价竞标，价格不透明，而且很多优质项目被设计公司垄断了。

99designs是固定价格套餐，价格透明，而且有严格的设计师审核机制。每个设计师都要提交作品集，平台会根据他们的获奖次数、客户好评数等打分，分为入门级、中级和顶级三个等级。

最关键的是，99designs的竞赛模式确保了你能看到多样化的设计方案。你不是在赌哪个设计师能做好，而是从一堆实实在在的作品里挑最满意的。这种体验完全不一样。

## 哪些人适合用这个平台

根据我的观察，以下几类人特别适合用99designs：

**创业公司**：刚起步，需要建立品牌形象，但预算有限。99designs能用可控的成本获得专业的设计。

**中小企业**：需要定期更新营销物料、产品包装等，可以建立一个设计师资源池，长期合作。

**独立创作者**：比如自己出书需要封面设计，举办活动需要海报，这些一次性设计需求在99designs上都能快速解决。

**对设计有想法但说不清楚的人**：如果你心里有个模糊的概念，但不知道怎么跟设计师描述，竞赛模式特别合适。你可以在看到不同设计方案后，逐渐明确自己到底想要什么。

## 怎么才能在99designs上得到好作品

说到这里，给大家分享几个使用技巧：

**写清楚设计需求**：99designs会引导你填写设计简介，包括你的品牌故事、目标受众、喜欢的风格、颜色偏好等。这些信息越详细，设计师就越容易抓住你的需求。

**给参考案例**：上传你喜欢的设计案例，或者用平台的风格选择工具，让设计师知道你的审美方向。

**及时给反馈**：竞赛期间，设计师会根据你的反馈不断调整作品。如果你能快速回复，告诉他们哪里好、哪里需要改，最终作品会更符合你的期望。

**选对套餐等级**：如果你的项目比较重要，比如公司Logo，建议直接选金牌或白金套餐，这样参与的都是顶级设计师，作品质量有保障。

**利用投票功能**：如果自己拿不定主意，可以用平台的投票功能，让团队或朋友帮你选。

## 关于退款和版权的问题

这两个是很多人关心的点，我专门查了平台的政策。

**退款政策**：99designs提供100%退款保证。如果竞赛结束后，你对所有提交的作品都不满意，可以在60天内申请全额退款。但有几种情况不能退款：设置了保证奖金的竞赛、已经进入最终回合的竞赛、已经选出获胜者的竞赛。

**版权归属**：一旦你选择了获胜设计并完成付款，该设计的全部版权就归你所有。平台会要求设计师签署版权转让协议，确保你可以无限制地使用这个设计。

这两点对于购买方来说是很重要的保障，基本上不用担心后续的法律纠纷。
